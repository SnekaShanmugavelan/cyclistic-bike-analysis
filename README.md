# cyclistic-bike-analysis
Cyclistic Bike Share Analysis

Project Objective

Analyze Cyclistic bike-share data to understand the behavioral differences between casual riders and annual members.

Tools Used

- Python
- SQL (BigQuery)
- Tableau
- Spreadsheet Analysis

Analysis Performed

- Data cleaning and preprocessing
- User type comparison
- Weekly ride usage analysis
- Average ride length analysis

Key Insights

- Casual riders tend to take longer rides.
- Annual members use bikes more frequently on weekdays.
- Casual riders are more active during weekends.
